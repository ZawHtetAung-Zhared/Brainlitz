# Getting Started

## npm install first

```
npm i
```

## start development with DEV config

```
npm start
```

## start development with STAGING config

```
npm run start:staging
```

## start development with LIVE config

```
npm run start:master
```

# Build and ~Test~

## DEV build

```
npm run build:dev
```

## STAGING build

```
npm run build:staging
```

## LIVE build

```
npm run build:master
```
